================================================================================
HEAT INDEX ALERT SYSTEM - REQUIREMENTS
================================================================================

HARDWARE
--------------------------------------------------------------------------------
- Raspberry Pi Pico 2 W
- DHT22 Temperature/Humidity Sensor
- 16x2 I2C LCD Display (I2C address 0x27)
- Breadboard and jumper wires
- Micro USB cable
- Power source (USB or 5V battery pack)


SOFTWARE & DOWNLOADS
--------------------------------------------------------------------------------

MicroPython Firmware:
- MicroPython v1.22+ for Raspberry Pi Pico 2 W
- Download: https://micropython.org/download/RPI_PICO2/


Required Library:
- lcd_i2c.py (I2C LCD driver)
- Download: https://github.com/dhylands/python_lcd


Project Files:
- main.py (from this repository)
- lcd_i2c.py (place in Pico root directory)


BUILT-IN LIBRARIES (No download needed):
- dht
- machine
- time
